Welcome to CellBender's documentation!
======================================

.. image:: /_static/design/logo_250_185.png
   :height: 185 px

*CellBender* is a software package for eliminating technical artifacts from high-throughput single-cell RNA sequencing (scRNA-seq) data.

For a brief survey of the package and its current capabilities, you can read
the :ref:`Introduction <introduction>`. For more comprehensive instructions
and quick-start tutorials, see the :ref:`Getting Started <getting started>` section.
